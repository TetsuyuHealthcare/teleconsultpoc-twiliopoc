# DP-Tele-Consultation

Video call app with twilio service.

## Technical Detail

* Backend:
  - NestJS
  - TypeORM (PostgreSQL)
  - Passport (JWT)
  - Bull + Redis (Background job)
  - Sendgrid (Email service)
  - ...

* Frontend:
  - ReactJS
  - Redux + Redux Toolkit
  - Antd
  - Axios
  - Twilio Video
  - ...
